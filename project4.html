Budget Tracker Web App

This project was a team-based project that simulated a real-world software engineering project. The goal was to develop a budget tracker web application to help users manage their income and expenses more efficiently. This web application would have transaction management capabilities that allow users to add income and expense transactions while the system automatically calculates their remaining balance and total cash inflows and outflows. The web app also categorized expenses into different categories (e.g., food, entertainment) to allow users to track their expenses for specific events. Additionally, the application would provide a transaction history feature that would enable users to review and track their past transactions.
