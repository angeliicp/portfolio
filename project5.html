Kaizen Management System

During my internship at PT Mesin Isuzu Indonesia, I was assigned to the HR department with the task of developing a Kaizen Management System. This web-based application was designed to digitalize the company's suggestion system program.

Responsibilities

  • Served as full-stack developer, implementing the MVC architecture
  • Implemented libraries such as DOMPDF, PHPMailer, and PHPOffice
    to enhance system functionality
  • Collaborated with HR department to gather
    requirements and conduct system reviews
  • Created user documentation to facilitate system adoption

Technologies Used

  • Code Igniter 3
  • MySQL and phpMyAdmin
  • Composer
  • Laragon
  • Visual Studio Code

Kaizen Management System is a digital platform where employees can submit their ideas for improving various aspects of the company's activities and facilities. Once submitted, these suggestions undergo an evaluation process that involves multiple levels of management.
